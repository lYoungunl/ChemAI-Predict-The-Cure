# ChemAI: Predict the Cure

Решение хакатона по предсказанию эффективности химических соединений против вируса.

## Задача

Предсказать три показателя для каждого химического соединения:
- **IC50** (mM) — концентрация, при которой вещество подавляет 50% активности вируса
- **CC50** (mM) — концентрация, при которой вещество токсично для 50% клеток
- **SI** (Selectivity Index) — индекс селективности (вычисляется как CC50 / IC50)

## Данные

- Обучающая выборка: 751 соединение, 210 признаков
- Тестовая выборка: 250 соединений

## Подход

### 1. Логарифмическое преобразование

Целевые переменные преобразованы в логарифмический масштаб:
log_IC50 = log1p(IC50)
log_CC50 = log1p(CC50)

### 2. Предобработка

- Пропуски заполнены медианой
- Признаки нормализованы (StandardScaler)

### 3. Модели

Обучены и сравнены 6 моделей:

| Модель | IC50 RMSE | CC50 RMSE |
|--------|-----------|-----------|
| LightGBM | **416.32** | **473.39** |
| XGBoost | 434.97 | 495.18 |
| RandomForest | 427.96 | 496.71 |
| GradientBoosting | 419.22 | 513.10 |
| KNN | 416.84 | 529.40 |
| SVR | 446.26 | 500.02 |

### 4. Ансамбли

- RF + GB + KNN: IC50 416.25, CC50 491.96
- RF + GB + KNN + LightGBM: IC50 415.36, CC50 483.43

## Результат

**Лучшая модель:** LightGBM

| Показатель | Значение |
|------------|----------|
| IC50 RMSE | 416.32 |
| CC50 RMSE | 473.39 |
